# Projeto de Mini-Jogos

Um projeto de três mini-jogos desenvolvidos em linguagem C.

**Desenvolvedores:** Jean Yves e Lucas Braz

## 📋 Jogos Inclusos

### 1. Perguntas e Respostas
Um jogo de 5 perguntas de conhecimento geral e sobre jogos com 4 alternativas cada.

### 2. Cobra na Caixa
Um jogo para dois jogadores onde cada um tenta encontrar o botão antes do outro, enquanto evita a cobra. O jogo possui 5 caixas, uma com um botão e outra com uma cobra.

### 3. Gousmas War
Uma guerra de estratégia onde dois jogadores controlam gousmas com medidor de fúria. Os jogadores podem atacar ou dividir fúria entre suas gousmas.

## 🛠️ Pré-requisitos

- Compilador C (GCC, Clang, etc.)
- Sistema operacional: Windows, Linux ou macOS
